The Menpo Project

Main Widgets

These are top-level functions for visualizig the various objects of the Menpo Project. They are separated into two categories:

  • Main Menpo Widgets
    These are the main widgets for visualizing objects of the menpo package.
  • Main MenpoFit Widgets
    These are the main widgets for visualizing objects of the menpofit package.

Note that these are the only functions that an end-user should use.
